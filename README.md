Game N2048

Cara Run Game:

g++ -g -O2 *.cpp -lncurses -o n2048

Maka akan muncul file n2048 yang mana adalah run file, 
Untuk menjalankan ketik: ./n2048

Cara Uninstall Game

rm -R -f n2048 n2048.dSYM

rm (remove)
-f (force argument)
